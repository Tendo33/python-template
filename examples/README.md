# Utils 使用示例

这个目录包含了项目中各个模块的使用示例。

## 文件说明

- `utils_example.py` - utils 工具包的使用示例

## 运行示例

```bash
# 运行 utils 示例
python examples/utils_example.py
```

## 示例内容

### utils_example.py

演示了 utils 包中的各种工具函数:

1. **日志功能** - 使用 loguru 进行日志记录
2. **装饰器** - 计时装饰器、重试装饰器等
3. **JSON 操作** - 读写 JSON 文件
4. **文件操作** - 文件复制、移动、哈希计算等
5. **日期时间** - 时间戳、日期格式化、时间计算
6. **字典操作** - 展平、合并、安全获取等
7. **列表操作** - 分块、批量处理等
8. **其他工具** - UUID 生成、邮箱验证等
