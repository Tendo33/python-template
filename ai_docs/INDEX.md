# AI Docs Index

## How to use this index

- 按任务场景找文档，不按文件名猜。
- 先看 `current/` 确认仓库今天真实长什么样，再看 `standards/` 确认默认做法。
- 涉及命令、目录结构、命名规则时，只引用 `reference/`。
